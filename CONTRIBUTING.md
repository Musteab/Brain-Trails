Welcome, fellow adventurer! We're glad you want to help improve the realm.

## How to Contribute
1. **Fork the repo** and create your branch from `main`.
2. **Setup your environment:** Follow the Quick Start guide in the README.
3. **Make your changes:** Add new features, slay bugs, or polish the UI. 
4. **Test your code:** Ensure your code builds (`npm run build`) and passes linting (`npm run lint`).
5. **Issue a Pull Request:** Describe your changes clearly!

## Code Style
- Use `npm run lint` before committing.
- Keep components modular.
- Use Tailwind for styling.
