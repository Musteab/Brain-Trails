# 🧠 Brain Trails

🧙‍♂️ **A gamified study companion with RPG elements**

Brain Trails turns your learning journey into an epic quest. Level up, earn gold, join a guild, unlock titles, and collect cosmetics as you study!

## ✨ Features
- 📜 **Interactive Notes:** Rich text editor with auto-save
- 🎴 **Flashcards:** Space repetition learning
- ⚔️ **Quests:** Daily challenges and achievements
- 🏪 **Shop:** Spend earned gold on frames, borders, and UI themes
- 🛡️ **Guilds:** Join forces with others
- 🎵 **Focus Mode:** Pomodoro timer with ambient Spotify player
- 🌙 **Atmosphere:** Gorgeous animated sun and moon backgrounds

## 🚀 Quick Start
```bash
git clone https://github.com/Musteab/Brain-Trails.git
cd Brain-Trails/brain-trails
npm install
```

Copy `.env.example` to `.env.local` and add your keys (Supabase + Google OAuth).

```bash
npm run dev
```

## 🛠️ Tech Stack
- **Frontend:** Next.js 14 (App Router), React, Tailwind CSS, Framer Motion
- **Backend/Auth:** Supabase, PostgreSQL
- **Icons & UI:** Lucide React, Shadcn (radix-ui)
