# Security Policy 🛡️

## Supported Versions
Currently, only the `main` branch (Beta) of Brain Trails is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| Beta    | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability 🚨
If you discover a security vulnerability within Brain Trails, please **DO NOT** open a public issue. 

Instead, please send an email directly to the maintainer or reach out via Discord. 

We take security seriously and will work to address any vulnerabilities responsibly and quickly. We will acknowledge your report within 48 hours and provide a timeline for a patch. 

Please include:
* A description of the vulnerability.
* Steps to reproduce the issue.
* Potential impact.
